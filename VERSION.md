# React Pull Refresh 更新日志

* 引入了 [react-hammerjs](https://github.com/JedWatson/react-hammerjs) 来更好的处理事件，以及组件 unmount 时卸载 hammer 实例

* 优化例子
* 增加了可自定义向下滑动刷新效果例子
* 修复部分 bug
* 完善文档
